Tugas ini bertujuan untuk menerapkan metode Regresi Linear Berganda dalam menganalisis faktor-faktor yang memengaruhi jumlah pernikahan di setiap kabupaten/kota Provinsi Sulawesi Tengah. Melalui data yang mencakup jumlah penduduk dan tingkat pengangguran terbuka menurut kabupaten/kota tahun 2018–2025, dilakukan proses pengumpulan data, eksplorasi data, pembersihan data (preprocessing), serta pembangunan model regresi untuk memprediksi jumlah pernikahan. Hasil analisis diharapkan dapat memberikan informasi mengenai pengaruh masing-masing variabel terhadap jumlah pernikahan sehingga dapat menjadi dasar pengambilan keputusan terkait kebijakan kependudukan dan ketenagakerjaan di setiap kabupaten/kota.
